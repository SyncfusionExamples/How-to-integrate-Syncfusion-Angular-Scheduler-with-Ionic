# Ionic Angular Integration with Syncfusion Scheduler

This project is an Ionic Angular application built using Standalone Components and integrated with the Syncfusion Scheduler. It provides calendar UI with support for multiple views, event creation. The use of Ionic ensures responsive layouts and touchâ€‘optimized interactions, making the scheduler ideal for mobile and tablet devices.

## Prerequisites
Ensure the following tools are installed:
- Node.js 16+
- npm 8+
- Ionic CLI
- Angular 17+
##  Installation
1. **Clone the project**.
2. **Install dependencies**
    ```bash
    npm install
    ```
##  Running the Application
Start the **Ionic server** using the below command.
```bash
ionic serve
```
